## stylelint-plugin-inactive-css

## Usage

```
npm i stylelint stylelint-plugin-inactive-css
```

## Setup

```
//.stylelintrc
{
  "plugins": [
    "stylelint-plugin-inactive-css"
  ],
  "rules": {
    "inactive-css/flex": true
  }
}
```
